# Remote Servers Resource Monitoring
This project is intended to build a utility tool for monitoring Resource  usage a remote Windows and linux server environmet
## Languages used:
- Node js and Express.js htt serve
- Shell scrips for windows written in powershell
- Shell scripts for Linux written in bash

The untility can be deployed as a web api services on any environment  and be interfaced with an external Web Application.
BY calling the API endpoint /systeminfo will respond with all the Resource ussage information
